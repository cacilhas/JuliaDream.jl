# JuliaDream.jl

I suggest you do not `Pkg.add()` on this source.

Rather, prefer git-clone this repo, install the deps, and run the `main.jl`
script.

This is NOT a lib: this is a study example.

## Dependencies

```julia
Pkg.add("AbstractTrees")
Pkg.add("Requests")
Pkg.add("Gumbo")
Pkg.add("Shoco")
```
